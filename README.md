## Getting Started

### Prerequisites

> python version 3.10

### Installation

clone this project

```
git clone https://github.com/albertparlys/final-project-albert-parlys-pyt4-proa-dts-2022.git
```

install depedencies

```
cd final-project-albert-parlys-pyt4-proa-dts-2022
pipenv install --requirements.txt
```
